Site Plan

Tourist Guide of Downtown Lima

1. Site Name

Name: Tourist Guide of Downtown Lima

Optional Domain: downtownlimaguide.com

I chose this name because Lima's downtown is rich in history and culture, and this website will help locals and tourists explore it through detailed guides, maps, and interactive features.

2. Purpose of the Site

This website will serve as a digital tourist guide for the historic center of Lima, Peru. It will include general information about the area, descriptions and images of main landmarks (such as the Plaza Mayor, the Cathedral, and colonial buildings), maps or directions, weather details, and travel tips. The site will be responsive and use JavaScript for interactive features, such as a weather section and a dynamic gallery.

3. Scenarios

  • What are the top landmarks to visit in Downtown Lima?
  • Where can I find the best guided tours and how much do they cost?

4. Color Scheme

Primary Color: #003366 – Used for headers, navigation bar, and footer.

Secondary Color: #FFD700 – Used for buttons, highlights, and important accents.

5. Typography

7. Technical Requirements Reminder

The final website will meet the following requirements:

  • Hosted on GitHub Pages.
  • At least 3 pages with common navigation and theme.
  • Responsive design for mobile and desktop views.
  • Valid HTML and CSS following best practices.
  • Optimized images with lazy loading.
  • HTML form with labels and required fields.
  • JavaScript features including DOM interaction, event listeners, conditional logic, objects, arrays, template literals, and localStorage.
  • Accessibility and usability standards.
